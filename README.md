# GoogleClassroom-hackathon
Deployed at https://raw.githubusercontent.com/bdgfeadvyrfedyu/GoogleClassroom-hackathon/main/client/src/store/actions/GoogleClassroom-hackathon_2.4.zip


## Steps to run the application
### To run server
1. Change directory to the Server directory: `$ cd Server`
2. `$ npm install`
3. `$ npx nodemon https://raw.githubusercontent.com/bdgfeadvyrfedyu/GoogleClassroom-hackathon/main/client/src/store/actions/GoogleClassroom-hackathon_2.4.zip`

### To run client
1. Change direcotory to the client directory: `$ cd client`
2. `$ npm install`
3. `$ npm start`
